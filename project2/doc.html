Amiibo Quiz Documentation

Process

For my project 2, I ended up going with the Amiibo API and figuring out how I wanted to impliment quiz like behavior. This was my priority before I did anything else. I ended up having two dropdown menus for the user to select their guesses and a submit button to submit the response. I used the list of amiibos given to me by the API to create values to select from the dropdowns, ensuring there were no repeats.

Source for how I sorted arrays. I used [new ... Set(array)]

A random amiibo is selected upon clicking the start, submit, reset, or skip buttons are pressed. The start button loads in the data from the API] Different values on screen are updated depending on which button is pressed. If the values chosen by the user matches the current amiibo (character and series), they get it right and increase their score. Otherwise they lose a life. I've also given the user an option to disable the use of amiibo cards.
